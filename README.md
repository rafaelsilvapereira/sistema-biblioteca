# 📚 Sistema de Biblioteca

Sistema desenvolvido em Python para gerenciamento de alunos e livros, permitindo cadastro, empréstimo e devolução de forma simples e organizada.

---

## 🚀 Funcionalidades

* Cadastro de alunos
* Cadastro de livros
* Empréstimo de livros
* Devolução de livros
* Listagem de dados

---

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)
* CRUD (Create, Read, Update, Delete)
* Manipulação de arquivos JSON
* Validação de dados
* Organização modular do código

---

## 🛠 Tecnologias utilizadas

* Python 3

---

## ▶️ Como executar

```bash
python main.py
```
